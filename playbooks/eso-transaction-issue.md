# Playbook: ESO Transaction Issue

**Trigger:** Something goes wrong with an Easy Street Offers transaction — wrong data in CITY, title issue, assignment delay, or partner communication breakdown.

---

## Step 1 — Check the Tracker
Open "Easy Street Acquisitions" (Google Drive). Find the property and confirm its current status.
This often provides the issue context without needing to escalate further.

## Step 2 — Identify the Stage

| Stage | Description |
|-------|-------------|
| Pre-contract | Property on ESO weekly list, not yet in CITY |
| In-contract | CITY intake in progress |
| Inspection | DV scheduled or completed |
| Assignment | Assignment Agreement in process |
| Closing | COE approaching or in progress |

## Step 3 — Route by Issue Type

**CITY issue (wrong data, label, system error):**
- Post in `#tech-support` Slack: property address, token, what's wrong, what it should be
- Tag CITY engineer (Agnes Omega) if ESO-workflow-specific
- If urgent → DM directly

**Title issue (EMD, HOA docs, outside title):**
- Email clientservices@osnational.com or teops@osnational.com
- Include property address and specific issue
- Loop in Acquisition Home Advisor (Jackson)

**Assignment timing at risk (approaching ≤7-day window):**
- Alert Program Manager immediately
- Calculate days remaining: COE date − today
- If <10 days and not yet sent → escalate now
- If <7 days and not executed → all-hands alert to PM, Acq HA, ESO

**ESO partner communication gap:**
- Email transactions@easystreetoffers.com
- If urgent → ESO leadership
- Document in "Easy Street Deals" process doc

## Step 4 — Document the Resolution
If the issue reveals a process gap:
- Update "Easy Street Deals" process doc
- Consider whether a CITY feature request would prevent recurrence

---

## Escalation

See [../programs/eso/escalation-playbook.md](../programs/eso/escalation-playbook.md) for full escalation paths.

Escalate to Mark Biggins when:
- Assignment timing cannot be recovered at operational level
- ESO partner relationship is at risk
- Systematic issue affecting multiple transactions
