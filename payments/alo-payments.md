# ALO Payments — Full Lifecycle

**Program:** Agent Led Offers
**Payment:** Per-closing payment to partner broker
**Vehicle:** Ramp PO

---

## Overview

When a partner broker successfully closes a deal through the ALO program, they receive a per-closing payment from Opendoor. This payment is an incentive for brokers who bring agents into the program and support successful closings.

---

## Full Payment Lifecycle

### Phase 1 — Closing Confirmed
- Closing is completed and confirmed in Opendoor systems
- Program Manager (or Melissa Giersz) receives notice of closing
- Entry is added or updated in "Agent Led Offers Payments 2026" (Google Drive)

### Phase 2 — Broker Verification
Before processing, confirm:
- [ ] Broker entity name (legal name for Ramp PO)
- [ ] Mailing address / payment details on file
- [ ] No prior payment for this same closing exists in tracker
- [ ] Closing date is confirmed

### Phase 3 — Ramp PO Creation
1. Open Ramp → Create new Purchase Order
2. **Payee:** Partner broker's legal entity name
3. **Amount:** Standard ALO rate (confirm current rate with program materials)
4. **Memo:** "This payment relates to our Agent Led Offers pilot. We are compensating partner brokers when a referred seller sells to Opendoor."
5. **Include property address in memo field** — this is critical for reconciliation

### Phase 4 — Submit and Confirm
- Submit PO for approval
- Confirm approval and payment execution
- Note Ramp PO number in the tracker

### Phase 5 — Tracker Update
In "Agent Led Offers Payments 2026" sheet:
- Payment date
- Ramp PO number
- Confirmation status
- Any notes (e.g., returned payment, address correction)

---

## Payment Issues

| Issue | Action |
|-------|--------|
| Broker says they didn't receive payment | Check Ramp status → verify address → reissue if needed |
| Can't match payment to a closing | Reach out to broker for property address |
| Duplicate payment risk | Cross-check full tracker before processing |
| Ramp PO stuck in approval | Follow up with approver directly |

Full escalation: [escalation-guide.md](escalation-guide.md)
