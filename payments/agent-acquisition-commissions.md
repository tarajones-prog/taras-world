# Agent Acquisition Commissions — Reference

**Programs:** HSA Agent Acquisition, ALO, referral programs
**What:** Commissions paid to agents who bring sellers to Opendoor

---

## Commission Types by Program

| Program | Commission Type | Trigger | Rate |
|---------|----------------|---------|------|
| HSA Agent Acquisition | 1% incentive | Agent-referred seller closes | ~1% of transaction |
| ALO (Agent Led Offers) | Per-closing payment | Broker's agent submits successful offer | Standard flat rate |
| Referral Program | Referral fee | Referred seller closes | Varies |

> Note: Specific rates are confirmed in current program materials. This is a framework reference.

---

## Common Rules Across All Commission Types

1. **Registration required** — Agent/broker must be registered before the referral
2. **Post-close payment** — All commissions paid after successful closing, not at contract
3. **Paid to broker entity** — Not to the individual agent (per brokerage structure)
4. **Address in memo** — Property address must be in any Ramp PO or payment memo
5. **Track before pay** — All payments recorded in tracker before processing

---

## HSA Agent Acquisition — 1% Incentive

This is the primary commission for the HSA Agent Acquisition training program.

- **Who earns it:** Agents who are enrolled in the program and refer sellers who close with Opendoor
- **Rate:** Approximately 1% of the purchase price (confirm with current program materials)
- **Paid to:** Agent's broker entity
- **Payment vehicle:** Ramp PO
- **Memo format:** [Property address] — HSA Agent Acquisition Program

For training materials on how HSAs explain this to agents, see [../programs/hsa/training-guide.md](../programs/hsa/training-guide.md).

## Payment Processing

All commission payments go through Ramp. Process:
1. Confirm eligibility (registration, close date, no duplicate)
2. Create Ramp PO with property address in memo
3. Submit for approval
4. Update tracker
5. Confirm delivery

Full Ramp guide: [ramp-process.md](ramp-process.md)
