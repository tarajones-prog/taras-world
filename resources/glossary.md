# Glossary — Opendoor and Real Estate Terminology

Reference for terms used across Opendoor Agent Partnership programs. Useful for onboarding, cross-team communication, and aligning with engineering or product partners.

---

## Opendoor-Specific Terms

| Term | Full Name | Meaning |
|------|-----------|---------|
| **ALO** | Agent Led Offers | Program where partner agents submit offers on Opendoor properties on behalf of buyers |
| **ESO** | Easy Street Offers | Acquisition partner — sources and assigns contracts to Opendoor |
| **CP** | Cash Plus | Opendoor product: cash offer + listing option for sellers |
| **CNML** | Cash Now More Later | Opendoor product (formerly Cash+): cash offer now + upside participation if property sells for more |
| **ELA** | External Listing Agent | Agents who list Opendoor-owned properties for resale |
| **MSRE** | Multi-Seller Real Estate | Program for ELAs managing multiple Opendoor listings |
| **HSA** | Home Sales Associate | Internal Opendoor role — front-line agent relationship managers |
| **CITY** | — | Opendoor's primary internal ops and transaction system |
| **Town** | — | Sometimes used interchangeably with CITY |
| **Admin** | — | admin.opendoor.com — Opendoor property administration portal |
| **ABAD** | Assignment Between Assignor and Buyer | Legal document formalizing contract assignment from ESO to Opendoor |
| **FSS** | Final Settlement Statement | Closing document showing all credits, debits, and fees at close of escrow |
| **FO** | Final Offer | Confirmed offer in CITY — required before intake can proceed |
| **DRI** | Directly Responsible Individual | The single person accountable for a task or program |
| **PSM** | Partner Success Manager | Team role managing partner relationships |
| **TC** | Transaction Coordinator | Manages transaction logistics from contract to close |
| **EP** | Experience Partner | Opendoor role (varies by context) |
| **OSN** | OS National | Opendoor's primary title and escrow partner |

## Real Estate Terms

| Term | Meaning |
|------|---------|
| **COE** | Close of Escrow — the closing date when ownership transfers |
| **EMD** | Earnest Money Deposit — good-faith deposit made when a contract is signed |
| **DV** | Diligence Visit — Opendoor's term for a property inspection |
| **BINSR** | Buyer's Inspection Notice and Seller's Response — the formal diligence repair negotiation document (common in AZ) |
| **MLS** | Multiple Listing Service — the database where agents list properties for sale |
| **RETS** | Real Estate Transaction Standard — protocol for accessing MLS data programmatically |
| **Assignable contract** | A purchase contract that can be transferred (assigned) from one buyer to another |
| **Assignment fee** | Payment made when a contract is assigned — ESO receives this from Opendoor at closing |
| **Escrow** | Neutral third party (OS National) that holds funds and documents until closing conditions are met |
| **HOA** | Homeowners Association — may have dues, documents, or approval requirements that affect closings |
| **Title** | Legal ownership documentation for a property |
| **CE** | Continuing Education — required for real estate license renewal |
