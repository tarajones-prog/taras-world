# MSRE — Multi-Seller Real Estate Program Overview

**Status:** Active — expansion markets
**Type:** ELA (External Listing Agent) / listing agent partnership program
**Last updated:** 2026-03

---

## What Is MSRE?

MSRE (Multi-Seller Real Estate) is a program where agents and brokers list multiple Opendoor-owned properties for sale. External Listing Agents (ELAs) are compensated for their listing and sales work.

This is a **resale program** — not acquisition. Opendoor already owns the properties; ELAs help sell them.
