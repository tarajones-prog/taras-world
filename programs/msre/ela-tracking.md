# MSRE — External Listing Agent (ELA) Tracking Process

---

## What We Track

For each active ELA agent/broker, track:
- Agent name and brokerage
- Markets they're assigned to
- Active listings under their management
- Days to Pending per listing
- Performance vs. benchmark
- Payment status and history

---

| Tool | Use |

---

## ELA Performance Metrics

**Primary metric:** Days to Pending
- Measures how quickly an ELA agent gets an Opendoor property under contract
- Top ELA agents consistently outperform on this metric
- Used to identify top performers for the Top ELA program

**Secondary metrics:**
- Number of active listings
- Close rate
- Agent responsiveness and communication quality

## Escalation

For ELA issues (agent non-performance, listing problems, payment disputes):
1. Check CITY for listing status
2. Review agent's performance history in tracking sheet
3. Direct communication with agent/broker
4. Escalate to Jackson (Top ELA) or Mark Biggins (program-level) as needed
