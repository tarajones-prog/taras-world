# HSA Agent Acquisition — Program Overview

**Status:** Active — training initiative in progress
**Type:** Internal-facing training and agent acquisition program
**Last updated:** 2026-03

---

## What Is This Program?

The HSA Agent Acquisition program trains Home Sales Associates (HSAs) on how to proactively engage agents in Opendoor's partner programs. The focus is on:
- Positioning the **CNML (Cash Now More Later)** product effectively
- Explaining the agent incentive structure (1% program)
- Increasing proactive outbound call volume to agents
- Building agent relationships that drive acquisition volume

---

## Why It Matters

HSAs are on the front lines of agent relationships. Well-trained HSAs who can clearly explain Opendoor's products and incentives drive more agent-referred acquisitions. This program is the foundation for that.

---

## Current Initiative

A formal training project is running in **Linear** (AGENT team) with deliverables due in early March 2026. Key outputs include:
- Training materials on CNML positioning
- Agent incentive explainer (1% structure)
- Proactive call volume guidance
- Scenario-based training document
