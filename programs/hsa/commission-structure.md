# Agent Acquisition Commission Structure

**Program:** HSA Agent Acquisition
**Incentive type:** Agent commission for referring sellers who close with Opendoor

---

## Overview

Opendoor pays a commission to agents who successfully refer sellers that close with Opendoor. This incentive is designed to motivate agents to proactively recommend Opendoor's products (particularly CNML) to their seller clients.

---

## Structure

| Component | Description |
|-----------|-------------|
| Incentive type | Percentage-based commission on closed transaction |
| Standard rate | Approximately 1% (confirm current rate with program materials) |
| Paid to | Agent's broker entity |
| Trigger | Successful close with Opendoor |
| Payment timing | After closing |
| Payment vehicle | Ramp PO or check |

---

## Eligibility

To receive a commission:
- Agent must be registered in the Opendoor agent partner program
- Referral must be documented before or at time of offer submission
- Transaction must close successfully
- Agent/broker must have valid payment information on file

---

## Payment Flow

1. Closing confirmed in Opendoor systems
2. Program Manager or payments team verifies eligibility
3. Payment processed via Ramp PO
4. Memo includes property address and program name
5. Payment sent to broker entity
6. Tracker updated with payment date and confirmation

---

## What Agents Need to Know

- Registration is required before submitting a referral
- Commission is paid to the **broker entity**, not the individual agent (depending on brokerage structure)
- Payment is made **after closing** — not at contract signing
- Questions go to the Partner Programs team

---

## Notes

- Commission rates are program-specific and subject to change
- This is a framework document — always confirm current rates with official program materials
- For payment disputes or non-receipt, see [../../payments/escalation-guide.md](../../payments/escalation-guide.md)
