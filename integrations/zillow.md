# Zillow — Partnership Context

**Type:** Real estate marketplace partnership
**Relevance:** Listing visibility, agent network, data context

---

## Overview

Zillow is the largest real estate marketplace in the US. Opendoor's relationship with Zillow is relevant to agent partnership programs in several ways:

- **Listing visibility:** Opendoor properties may appear on Zillow, affecting agent and buyer awareness
- **Agent network:** Many partner agents use Zillow's platform for listings and buyer leads
- **Market data:** Zillow's market data (Zestimate, days on market, listing history) provides context for pricing discussions with agents
- **Partnership context:** Opendoor and Zillow have had various partnership arrangements — relevant to understanding the competitive and cooperative landscape

---

## Relevance to Agent Programs

| Program | Zillow Connection |
|---------|------------------|
| ESO | ESO-sourced properties may appear on Zillow if listed before Opendoor acquisition |
| MSRE / ELA | ELA agents may use Zillow to market Opendoor resale inventory |
| ALO | Partner agents often source buyers through Zillow buyer leads |
| Cash Plus | CP listings visible on Zillow — affects agent and buyer awareness |

---

## Data Flows

The `opendoor-labs/rets` public repository (see [opendoor-labs-public-tools.md](opendoor-labs-public-tools.md)) provides a Real Estate Transaction Standard (RETS) client that can pull MLS/listing data — similar to how Zillow aggregates listing data from MLS feeds.
