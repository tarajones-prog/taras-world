## How to Add a New Integration

When a new partner or tool integration is added:
1. Create a new file in this folder: `[partner-name].md`
2. Document: what they do, how they connect to our programs, key contacts (by role), access instructions
3. Add a row to the table above
4. Link from any relevant program README
