# Zendesk — Support Ticketing

**What:** Zendesk is the support ticketing platform used for escalation tickets and formal issue tracking.

**Status:** As of early 2026, the team is migrating to Zendesk for escalations. Some workflows may still be in transition.

---

## When to Use Zendesk

Use Zendesk for:
- Formal escalation tickets that need tracking and SLA accountability
- Partner-facing issues that require a documented resolution path
- Any escalation that needs to be shared across teams

Use Slack for:
- Urgent, fast-moving issues where real-time communication matters
- Quick questions that don't need a formal ticket

---

## Escalation Ticket Template

See [../templates/escalation-ticket.md](../templates/escalation-ticket.md) for the standard ticket format.

Key fields in every ticket:
- Property address
- CITY token / TXN ID
- What's wrong vs. what it should be
- Impact (affects COE? affects commission? multiple transactions?)
- Owner / next action

---

## Transition Note

Zendesk adoption was still in progress as of early 2026. If you're unsure whether to use Zendesk or Slack for a specific escalation, default to Slack for urgency and create a Zendesk ticket for tracking.
